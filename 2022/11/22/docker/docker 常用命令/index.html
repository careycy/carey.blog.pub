docker 常用命令

查看docker常用命令：

查看本地有哪些镜像：

              -a   参数a = all  同时显示 当前镜像分层的镜像内容 (扩展 镜像可以包含多层内容)
              -q   显示镜像的id

显示镜像的备注信息： docker images –digests 主要显示参数 DIGEST 备注说明信息

从远程仓库获取镜像 docker pull 镜像名称

参数详解

　-v 本地目录:容器目录 或 -v 容器目录

示例：

将本机的当前目录，挂载到镜像中的/python目录　　

docker run -it 镜像id/镜像名称 – 创建镜像容器实例 it 为保持容器虚拟会话窗口
docker run -it –name 自定义镜像名称 镜像名/镜像id

docker ps — 查看当前正在运行的容器 参数 -a 查看所有创建的容器实例包括停止的 “ -n 参数“返回特定行数

exit 在容器中执行，可以正常关闭当前运行的容器实例

Ctrl + p +q 容器内执行，可以退出当前容器，并保持容器正常运行

docker start 容器实例id – 可以启动已经关闭的容器实例

docker restart 容器实例id –可以重启制定的容器实例

docker stop 容器实例id –可以在容器外停止容器的运行

docker rm -f $(docker ps -a -q) – 强制删除正在运行的容器实例 ，括号内为表达式，返回所有的容器实例id

docker attach 容器id – 根据提供的容器id返回正在运行的容器当中

docker exec -it 容器id /bin/bash 执行容器中的命令行

docker exec -t 容器id ls -l /tmp –在镜像外发起shell命令执行对镜像的操作

docker cp 容器id:/tmp/yum.log /root 从容器中拷贝指定的文件内容到宿主机中
